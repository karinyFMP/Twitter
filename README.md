# Projeto Twitter Clone - Avaliação G2 DSPW

Este projeto consiste em um clone simplificado do Twitter desenvolvido para a avaliação prática da disciplina de Desenvolvimento de Software para Web. A aplicação contempla um sistema completo de autenticação e um feed interativo de posts com sistema de curtidas.

## 👥 Integrantes da Dupla
* **[Matheus Carvalho Pina]**
* **[Kariny Ferreira de Melo]**

## 🛠️ Tecnologias Utilizadas
* **Frontend:** React, React Router Dom, React Hook Form, Axios.
* **Backend:** Node.js, Express, JWT (JSON Web Tokens), Bcrypt (para hash de senhas).
* **Banco de Dados:** SQLite.

## 🚀 Instruções de Como Rodar a Aplicação

Siga os passos abaixo para executar o projeto localmente em sua máquina.

### 1. Pré-requisitos
Certifique-se de ter o [Node.js](https://nodejs.org/) instalado em sua máquina.

### 2. Clonar o Repositório
